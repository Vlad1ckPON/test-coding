Множества - просто набор элементов - {}
A = {x1, x2, x3}
Ну тут ты знаешь ∈ - принадлежит в чему либо и ∉ - не пернадлежит
Также переменные могут обладать свойствами, например
Этот знак ⊂ знак что данное можество является подмножетсвом другого
Конкотенация или объединение множеств - ∪
Знак пресечения множеств - ∩ 
∅ - Знак пустого множества
\ - Знак разницы множеств

Какие множества существуют:
N - множество натуральных чисел {1, 2, 3, 4 ...}
Z - множество целых чисел {0, +-1, +- 2, +-3, +-4, ... }  
Q - множество рациональных чисел - множество дробей, которые состоят из целых чисел {x ∈ R: p/q, q ∈ Z}
R - множество вещественных(дейтвительные) чисел
C - множество компклексных чисел
Кстати N ⊂ Z ⊂ Q ⊂ R ⊂ C

Функции одной пременной
Функция - соотвествие элементов одного множества в элемента другого множества.(f: X -> Y)
Например: множество A = {1, 2, 3}, B = {1, 4, 9}, если придумать правило при помощи которго можно эти элементы к друг другу сопоставить,
то будет функция f(x) = x ** 2, типо лямбда
Чтобы не путать множества в функциях их называют: множество, элементы которого трансформируются в элементы другого множества, называется - 
область определения, а множества, элементы которого для которого трансформируются элементы области опредения, называется - область значения 
В функцию можно передать любое множество
В итоге получется кривая линия, которая соединяет все(бесконечность) точек функции на графике
Формулировка f: R -> R, всегда будет число от нуля до +бесконечности - f: R -> (0, +inf)
g =(x) = 1/x, g: R -> R - тоже Формулировка с запасом, тк туда в место x нельзя подставить 0, также как и g = 0 
Область определения и область значения функции g - D(g) = (-inf, 0) U (0, +inf), E(g) = (-inf, 0) U (0, +inf)

Образ функции
Образ множества A для функции f - это множество значений, которые принимают f при условии x ∈ A
Образом области определения функции является область значения A = d(f) - E(f)
() - крайние числа не включаются, [] - крайние числа включаются
f([1, 2]) = [1, 4](образ можно например посмотреть на графике, [1, 2] будет x, а те y которые подходят будут образом) 

Прообраз Функции 
Тут тоже самое, только наоборот прообраз будет определяться у области значения, и соотвественно будет подстовляться на графике в y, 
и находиться через подходящие
