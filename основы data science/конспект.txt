Для начала, прежде чем выбирать и обучать модель, нужно сначала сделать вывод о данных, для этого есть модуль matplotlib,
В котором можно посмотреть на данных через гистограммы, графики, диаграммы и тд (остальная информация по функция в файле с названием библиотеки)
Столбчатые графики подходят для подходят, когда нужно показать варьируемость некоторой величины среди некоего дискретного множества элементов
Также его можно использовать, как гистограмму, то есть график распределения значений
Третий аргумент в методе bar задает ширину столбцов, метод axis диапазон значений по осям
График считается плохим, если он начинается не с нуля, потому что легко может дезориентировать других людей
Линейные графики подходят для отображения трендов
Диаграммы рассеивания подходят для визуализации связи между спаренными множествами данных
